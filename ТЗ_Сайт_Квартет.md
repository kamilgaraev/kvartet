# Техническое задание: Сайт рекламного агентства "Квартет"

## 1. Общее описание проекта

### 1.1 Цели проекта
- Создание современного, SEO-оптимизированного сайта рекламного агентства "Квартет"
- Увеличение конверсии и привлечение новых клиентов
- Автоматизация процессов управления контентом через админ-панель
- Улучшение позиций в поисковых системах (Яндекс, Google)

### 1.2 Целевая аудитория
- Малый и средний бизнес в Уфе и Республике Башкортостан
- Организации, нуждающиеся в рекламных услугах
- B2B клиенты
- Частные предприниматели и стартапы
- Крупные корпорации, нуждающиеся в комплексном рекламном обслуживании

### 1.3 Информация о компании
- **Название**: Производственно-рекламная группа "Квартет"
- **Местоположение**: г. Уфа, ул. Ленская, 128
- **Направление**: Рекламное агентство полного цикла
- **Опыт работы**: Многолетний опыт в сфере рекламы

## 2. Технический стек (SEO-оптимизированный)

### 2.1 Frontend
- **Next.js 14** (App Router) - для SSR/SSG и идеального SEO
- **TypeScript** - типизация и качество кода
- **Tailwind CSS** - современная стилизация
- **Framer Motion** - анимации и интерактивность
- **React Hook Form** - управление формами
- **Zod** - валидация данных

### 2.2 Backend
- **Next.js API Routes** - серверная логика
- **Prisma ORM** - работа с базой данных
- **PostgreSQL** - основная база данных
- **NextAuth.js** - аутентификация для админ-панели

### 2.3 SEO и Performance
- **next-sitemap** - автогенерация sitemap.xml
- **next-seo** - мета теги и структурированные данные
- **Sharp** - оптимизация изображений
- **Cloudinary** - CDN для медиафайлов
- **Google Analytics 4** - аналитика
- **Yandex.Metrica** - российская аналитика

### 2.4 Хостинг и инфраструктура
- **Vercel** - основной хостинг (оптимален для Next.js)
- **Supabase** или **PlanetScale** - база данных
- **Cloudinary** - хранение изображений
- **Telegram Bot API** - уведомления о заявках

## 3. Структура сайта

### 3.1 Публичная часть
```
├── Главная страница (/)
├── Услуги (/services)
│   ├── Наружная реклама (/services/outdoor)
│   ├── Полиграфия (/services/printing)
│   ├── Интерьерная реклама (/services/interior)
│   └── Брендинг (/services/branding)
├── Портфолио (/portfolio)
├── О компании (/about)
├── Контакты (/contacts)
├── Блог (/blog) - для SEO
└── Калькулятор стоимости (/calculator)
```

### 3.2 Админ-панель (/admin)
```
├── Авторизация
├── Dashboard
├── Управление услугами
├── Управление портфолио
├── Управление блогом
├── Заявки клиентов
├── SEO настройки
└── Аналитика
```

## 5. SEO-требования

### 5.1 Техническое SEO
- **Core Web Vitals**: FCP < 1.8s, LCP < 2.5s, CLS < 0.1
- **Lighthouse Score**: 95+ для всех метрик
- Семантическая HTML разметка
- Правильная структура заголовков (H1-H6)
- Микроразметка Schema.org (Organization, Service, Review)
- robots.txt и sitemap.xml
- Канонические URL

### 5.2 Контентное SEO
- Уникальные Title и Description для каждой страницы
- Alt-теги для всех изображений
- Внутренняя перелинковка
- Хлебные крошки
- Оптимизация под ключевые запросы:
  - "реклама Уфа"
  - "рекламное агентство Уфа"
  - "печать баннеров Уфа"
  - "наружная реклама Башкортостан"

### 5.3 Локальное SEO
- Google My Business интеграция
- Яндекс.Справочник
- Структурированные данные LocalBusiness
- Отзывы на картах
